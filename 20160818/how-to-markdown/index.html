Markdown是什么？

Markdown is a text-to-HTML conversion tool for web writers.
Markdown 是一种轻量级的标记语言，目前我写这篇文章的时候也是在使用Markdown，使得我很简单地就完成了文章的一些排版还有文字标记，使得文章显得直观美观可读性高

使用 Markdown 的优点

  • 专注你的文字内容而不是排版样式。
  • 轻松的导出 HTML、PDF 和本身的 .md 文件。
  • 纯文本内容，兼容所有的文本编辑器与字处理软件。
  • 可读，直观。适合所有人的写作语言

Markdown用法

下面列举一些我平时经常遇到的一些问题和一些经常忘记的用法，至于一些基本的语法或者更高级的应用可以去看看参考
1.Markdown原文里面缩进在渲染后都没有了，所以用 &#160;或者&nbsp; 表示空格
2.Markdown特殊符号的显示
Markdown是HTML标记的一部分，所以使用起来会受到HTML的一些限制
比如<>可以正常显示，但是类似<abc>的符号就会变成html标签了，想正常显示最好写在两个反引号``或者代码块里面
如果你要在正文用，请使用HTML的编码写
< &lt;
<= &le;
> &gt;
>= &ge;
至于其他特殊符号请参考:HTML特殊符号对照表 - 站长工具
其他转义后可以显示的字符：
\\ 反斜杠
\* 星号
\_下划线
\{\} 大括号
\[\] 中括号
\(\) 小括号
\# 井号
\+加号
\- 减号
\. 英文句号
\! 感叹号
\ ` 反引号

3.上标和下标
上标 216 2<sup>16</sup>
下标 216 2<sub>16</sub>

4.超链接
超链接 [百度]( http://www.baidu.com)
